## Getting Started

### Prerequisites

Install NPM

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/carinadiesel-dev/calico-creatives-react.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
